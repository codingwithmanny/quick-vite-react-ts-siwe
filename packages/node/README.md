# Node TypeScript Bootstrap

Backend API for wallet verification.

## Requirements

- NVM or NodeJS `v16.14.0`

## Local Setup

While in project directory:

### 1 - Install Depencies

```bash
yarn install; # npm install;
```

### 2 - Server Start

`Development:`

```bash
yarn dev; # npm dev;
```

`Production:`

```bash
yarn start; # npm start;
```

## Production Commands

`Build`

```bash
yarn build; # npm run build
```

`Build & Serve`

```bash
yarn start; # npm start
```

## Tests

`All Tests`

```bash
yarn test; # npm run test;
```

`Jest Watch`

```bash
yarn test:jest; # npm run test:jest;
```

`Jest Coverage`

```bash
yarn test:coverage; # npm run test:coverage;
```

`Eslint`

```bash
yarn test:lint; # npm run test:lint
```
