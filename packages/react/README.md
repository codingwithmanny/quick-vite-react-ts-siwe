# Vite React TypeScript

Frontend interaction for wallet verification.

## Requirements

- NVM or NodeJS `v16.14.0`

## Local Setup

While in project directory:

### 1 - Install Depencies

```bash
yarn install; # npm install;
```

### 2 - Server Start

`Development:`

```bash
yarn dev; # npm dev;
```

`Production:`

```bash
yarn start; # npm start;
```

## Production Commands

`Build`

```bash
yarn build; # npm run build
```

`Build & Serve`

```bash
yarn start; # npm start
```
